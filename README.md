# 💼 Portfolio Website — Santosh Kumar

> **A fully animated, responsive personal portfolio built with pure HTML5, CSS3 & Vanilla JavaScript.**

🔗 **Live Website:** [View Portfolio](https://rajputnikusingh.github.io/portfolio-website)

---

## ✨ Features

| Feature | Details |
|---------|---------|
| 🎯 Custom Animated Cursor | Smooth dual-layer cursor with hover effects |
| 🌌 Particle Canvas Background | 80+ animated particles with connected lines |
| ⌨️ Typing Animation | Auto-typing role descriptions with backspace effect |
| 🎚️ Animated Skill Bars | Scroll-triggered progress bars for each skill |
| 📜 Scroll Reveal Animations | Elements fade in as you scroll down |
| 📱 Fully Responsive | Works perfectly on mobile, tablet & desktop |
| 🔢 Counter Animation | Live number count-up on load |
| 🎨 Dark Futuristic Theme | Deep space aesthetic with accent colors |
| 🧭 Smooth Navigation | Fixed nav with mobile hamburger menu |
| ⬆️ Scroll-to-Top Button | Appears after scrolling down |

---

## 🛠️ Tech Stack

- HTML5      — Structure & Semantics
- CSS3       — Styling, Animations, Responsive Layout
- JavaScript — Particle Canvas, Typing Effect, Scroll Animations
- Google Fonts — Syne, Space Mono, Inter

---

## 📁 Sections

- **Hero** — Name, typing animation, code card, CTA buttons, stats
- **About** — Personal info, highlights, detail table
- **Skills** — Animated skill cards with progress bars
- **Projects** — Live project cards with tech badges & links
- **Education** — Animated timeline
- **Contact** — All social links + availability card
